## üéÆ Features
- **Physics-Based Puzzles**: Use gravity and momentum to squash dinosaurs.
- **Limited Shot Challenge**: Try to eliminate all dinos with a limited number of rock drop.
- **Unreal Engine 5**: Updated to run on the latest engine technology.
- **Multiple Levels**: Different environments with unique puzzle layouts.

## üõ†Ô∏è Technical Details

This project uses **Unreal Engine 5.7**. It is a Blueprint-based project structured for mobile and desktop deployment.

### Dependencies
- **Unreal Engine**: `5.7`

### Setup & Installation
1. Clone the repository.
2. Improved the `.uproject` association if needed (Right-click `DinoHunt.uproject` > "Switch Unreal Engine Version..." > Select 5.5).
3. Double-click `DinoHunt.uproject` to open in the Unreal Editor.
4. Press **Play** in the editor to start squashing.

### Project Structure
- **Content/Maps**: Level files.
- **Content/Textures**: Game assets and UI images.
